# <%= name %>

A <%= name %> does something special.


## Example

```jsx
<<%= name %> />
```

## Props

| Prop            | Type       | Default     | Description                     |
| --------------- | ---------- | ----------- | ------------------------------- |
| className       | `string`   |             | The className of the component. |
| innerRef        | `Function`  |            | Retrieve the inner DOM node.    |
