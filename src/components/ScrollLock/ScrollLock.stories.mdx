# ScrollLock

ScrollLock is component which prevents scrolling past the top or bottom of a container. Wrap your scrollable element in a ScrollLock to use it.

## Browsers

Under the hood it applies the css rule [`overscroll-behavior`](https://developer.mozilla.org/en-US/docs/Web/CSS/overscroll-behavior) to add enable this behaviour, unfortunately it doesn't work on safari yet.

### Safari

For Safari we use an event handler on the `wheel` event, is less performant but it only applies on this browser.
