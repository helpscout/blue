# AvatarStack

An AvatarStack component displays an array of `Avatar`. This component has a max number of Avatars it will render. Any additional Avatars will be consolidated into a `+` counter that appears at the end.

<Preview>
  <Story name="variation: AvatarStack">
    <AvatarStack max={5}>
    {AvatarSpec.generate(5).map(avatar => {
      const { name, image, status } = avatar
      return <Avatar image={image} key={name} name={name} status={status} />
    })}
  </AvatarStack>
  </Story>
</Preview>
