# TypingDots

A TypingDots component is a bouncing ellipsis animation, typically used in chat/messaging based UI.

## Example

```jsx
<TypingDots />
```

## Props

| Prop      | Type     | Default | Description                     |
| --------- | -------- | ------- | ------------------------------- |
| className | `string` |         | The className of the component. |
