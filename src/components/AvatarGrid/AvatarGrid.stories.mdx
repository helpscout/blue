# AvatarGrid

An AvatarGrid component displays an array of `Avatars` in a grid-like manner. This component has a max number of Avatars it will render. Any additional Avatars will be consolidated into a `+` counter that appears at the end.


<Preview>
  <Story name="variation: AvatarGrid">
    <div style={{ background: '#eee', padding: 10 }}>
    <AvatarGrid borderColor="#eee" showStatusBorderColor max={14}>
      {AvatarSpec.generate(20).map(avatar => {
        const { name, image, status } = avatar
        return (
          <Avatar
            image={image}
            key={name}
            name={name}
            shape="rounded"
            status={status}
          />
        )
      })}
    </AvatarGrid>
  </div>
  </Story>
</Preview>
