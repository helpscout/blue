# ControlGroup

This component is used as a wrapper to create interfaces where Control-based components (like `Input`) can be combined.

## ControlGroup.Item

This component renders a Control-based component within a `ControlGroup`. It is used to combine several Control-based components together.
This component's width is determined by it's child content.
