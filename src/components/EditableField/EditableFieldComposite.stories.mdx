# EditableFieldComposite

This component allows the grouping and in-lining of multiple single value Editable Fields.

`EditableFieldComposite` is in essence a Higher Order Component that decorates its children to enable grouping them.

<Preview>
  <Story name="EditableField Composite">
    <ContainerUI>
      <EditableFieldComposite placeholder="Add a name">
        <EditableField
          label="First Name"
          name="first_name"
          type="text"
          placeholder="First Name"
          value="Johnny"
        />
        <EditableField
          label="Last Name"
          name="last_name"
          type="text"
          placeholder="Last Name"
          value="Cash"
        />
      </EditableFieldComposite>
    </ContainerUI>
  </Story>
</Preview>
