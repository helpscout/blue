# Flexy

A Flexy component is a layout-based component that uses Flexbox to intelligently align elements together. Flexy can be helpful to vertically align UI elements of varying heights.

## Flexy

A Flexy.Item component is a layout-based component used within `Flexy`. This component takes up as much space as it's content requires.

## Block

A Flexy.Block component is a layout-based component used within `Flexy`. This component fills up space between or around `Flexy.Item` components.
